# csd-javascript

## SETUP locally

### Clone the repo

git clone `https://github.com/jensen-und-komplizen/csd-javascript`

### Check the `node` version

run `node -v` in the terminal (16.14.2)

### Install dependencies

run `npm install` in the console

### Start the app

`npm install`
